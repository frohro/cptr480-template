# AI Reflection Journal

One entry per week, in `weekN.md`. Each entry should be 1–2 paragraphs (informal — graded on completion, not quality).

Suggested questions to answer each week:
- What did you ask Copilot? (specific prompts are more useful to write down than vague descriptions)
- What did it get right that saved you time?
- What did it get wrong, and how did you catch it?
- Did using Copilot help you understand the material better, or did it short-circuit your thinking?
- If you could redo this week, would you use AI differently?

The goal of this journal is to make you a *deliberate* AI user — aware of when it helps and when it misleads.
